# Running

To run the API the following arguments needs to be passed:

```
FB_APP_ID=APP_ID \
INITIALS_BLACKLIST_ID=ID \
FB_STORAGE_BUCKET=FIREBASE_BUCKET_NAME \
dart_frog dev
```
